## Project Bonsai

[Project Bonsai](https://docs.microsoft.com/bonsai/?wt.mc_id=eventspg_16482_webpage_reactor) is a Low-code AI development platform. Improve production and reduce downtime with engineer-built AI that gives optimization guidance or makes independent decisions. Join Microsoft Cloud Advocate Josh Ndemenge for these 4 live streams to learn everything you need to know.
